# FlowPilot

FlowPilot is a portfolio-grade B2B team workspace and task management SaaS built as a monorepo.

## Apps

- `frontend`: Next.js application
- `backend`: Express API with Prisma and PostgreSQL

## Getting Started

Install dependencies from the repository root:

```bash
npm install
```

Run each app separately:

```bash
npm run dev:frontend
npm run dev:backend
```

## Current Status

This repository currently contains:

- monorepo workspace foundation
- backend service scaffold
- Prisma schema for the MVP domain model
- database migration and seed scaffolding
